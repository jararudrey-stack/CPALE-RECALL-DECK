CPALE Recall Deck v5.6

This is a redone countdown/clock update based on the stable v5.4 build.

Countdown redesign:
- Flip-card inspired countdown based on the reference image.
- Countdown shows only DAYS and HOURS.
- Soft yellow flip cards are blended with the app's dark glass aesthetic.
- The date line remains:
  Today is Sunday, July 5
- Added a working real-time Philippine clock.
- Added a sun/moon toggle beside the clock:
  - Sun from 6:00 AM to 5:59 PM
  - Moon from 6:00 PM to 5:59 AM
- Date, clock, and toggle use Asia/Manila timezone and refresh every second.

How to update GitHub:
1. Download and extract this ZIP.
2. Replace these files:
   - index.html
   - manifest.webmanifest
   - service-worker.js
   - README.txt
3. Commit changes.
4. Open:
   https://jararudrey-stack.github.io/CPALE-RECALL-DECK/?v=56

If Safari still shows the old version, refresh several times or clear website data for the GitHub Pages site.
